# 日文学习系统 (JapStudySystem)

一个基于 Java Swing 的日文单词与语法学习工具，支持卡片式浏览、随机测试、掌握度跟踪，以及通过 DeepSeek 进行图片识别批量导入。

## 功能特性

- **卡片式浏览**：以卡片样式展示所有已保存的单词和语法，支持滚动查看
- **添加考察点**：手动新增日语单词或语法条目，语法条目支持例句和例句译文
- **模糊搜索**：输入日文或中文关键词进行查找，支持在线编辑和删除
- **随机测试**：基于掌握规则的智能抽查，自动跟踪学习进度
- **图片识别**：通过 DeepSeek 识别图片中的日语内容，批量解析并导入词库
- **数据持久化**：退出时自动保存所有学习数据和测试记录
- **内置 JDK**：无需安装 Java 环境，解压即用

## 使用方法

### 启动程序

1. 将 `JapStudySystem` 文件夹完整解压到任意位置（建议路径不含中文和空格）
2. 确保 `JapStuSystem启动.bat`、`bin/`、`jdk-24/` 在同一文件夹下
3. 双击 `JapStuSystem启动.bat` 即可启动

> 无需手动安装 Java 环境，程序使用内置的 JDK 24 运行。

### 主要功能

#### 查看全部

以卡片样式展示所有已保存的单词和语法。每张卡片显示类型、日文、释义、掌握率；语法卡片额外显示例句和例句译文。卡片上可直接点击**修改**或**删除**。

#### 添加考察点

1. 点击**添加考察点**
2. 输入日文，点击**确认输入**
3. 输入中文释义，点击**确认输入**
4. 选择类型（**单词** / **语法**）
   - 单词：直接添加完成
   - 语法：继续输入例句和例句译文后完成添加

#### 查找考察点

输入日文或中文关键词进行模糊搜索，找到后可进行**修改日文**、**修改释义**、**修改例句**、**修改例句译**、**删除**等操作。

#### 随机测试

系统随机抽取未完全掌握的内容进行记忆测试：

1. 点击**随机测试**
2. 看到日文表达后，点击**显示答案**查看释义和例句
3. 根据记忆情况点击**记得**或**不记得**
4. 达到掌握要求后，该条目自动从待学列表移除

#### 图片识别（DeepSeek）

1. 点击**图片识别**，程序自动打开 DeepSeek 网页并复制识别指令
2. 在 DeepSeek 上传包含日语内容的图片
3. 粘贴指令并发送，复制 DeepSeek 的回复
4. 回到程序，粘贴回复并点击**解析并预览**
5. 在预览卡片中确认/修改条目，点击**全部添加到词库**

## 掌握规则

| 阶段        | 条件            | 所需正确次数 |
| ----------- | --------------- | ------------ |
| 初始        | 开始学习        | 5 次         |
| 第 1 次出错 | 累计答错 2 次   | 6 次         |
| 第 2 次出错 | 累计答错 4 次   | 7 次         |
| ...         | 每累计答错 2 次 | +1 次        |

当正确次数达到动态要求时，该条目自动视为已掌握，不再出现在随机测试中。

## 数据说明

所有学习数据保存在 `D:/JaStu.txt`，包括：

- 单词/语法条目（日文、中文释义、类型、例句、例句译文）
- 测试次数（examTimes）
- 正确次数（trueTimes）

> 正常点击**保存退出**关闭程序，下次打开时数据完整保留。

## 常见问题

### 双击启动脚本没反应或闪退

检查是否完整解压所有文件，`JapStuSystem启动.bat`、`bin/`、`jdk-24/` 应在同一文件夹下；确认文件夹路径不含中文或特殊符号。

### 窗口大小不合适

直接拖动窗口边缘自由拉伸调整大小，卡片会自动换行排列。

### 图片识别时浏览器未自动打开

手动访问 [https://chat.deepseek.com/](https://chat.deepseek.com/)，并将识别指令粘贴到对话框中。

### 解析内容格式不正确

请确保 DeepSeek 回复严格按以下格式（制表符分隔）：

```
日文表达[TAB]中文释义[TAB]类型[TAB]例句[TAB]例句翻译
```

若个别条目格式异常，可在预览卡片中手动修改后再添加至词库。
